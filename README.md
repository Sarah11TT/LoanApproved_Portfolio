## 📌 Project Overview
This project addresses a critical banking challenge: **Automating credit risk assessment while maintaining high transparency.** Using a dataset of 5,000 applicants, I built an end-to-end pipeline that cleans financial data, identifies key risk drivers, and deploys a Machine Learning model for real-time approval predictions.

## 🛠️ Technical Stack
- **Languages:** Python (Pandas, NumPy, Scikit-Learn)
- **Visualization:** Plotly, Streamlit
- **Machine Learning:** Random Forest, Gradient Boosting, Decision Trees
- **DevOps:** Git, GitHub, Streamlit Cloud

## 🚀 Key Features
1. **Automated ETL Pipeline:** Modular cleaning script handling missing data and financial outlier correction (Clipping).
2. **Advanced ML Modeling:** A comparison of three ensemble models with **Stratified Splitting** to handle class imbalance.
3. **Executive Dashboard:** Interactive KPIs and Sunburst charts for stakeholder insights.
4. **AI Predictor:** A user-friendly form that calculates loan approval probability based on the best-performing model.

## 📈 Business Insights
- **Credit Score Correlation:** Analysis revealed that the 'Good' credit score threshold (670) is the strongest predictor of approval, increasing probability by 38%.
- **Risk Indicators:** High Debt-to-Income (DTI) ratios (>0.6) significantly correlate with loan denials even in high-income segments.

## 📂 How to Run

3. Run the app: `streamlit run app.py`
